1. Первый инстанс должен быть установлен через DirectumLauncher. Для этого необходимо:
  1.1. Запустить DirectumLauncher из папки экземпляра системы;
  1.2. Открыть раздел "Настройка", в блоке variables задать новую переменную instance_name: '<код экземпляра>';
  1.3. Проверить другие переменные в блоке variables (http порт, имя хоста, протокол);
  1.3. Нажать "Сохранить"! Иначе система будет установлена в режиме одного экземпляра;
  1.4. Вернуться в раздел "Установка", название точки обмена Rabbit должно совпадать с кодом экземпляра;
2. Для развертывания второго инстанса необходимо:
  2.1. Разархивировать DirectumLauncher в папку где будет установлена система;
  2.2. Скопировать в папку архивы Development Studio, Deployment Tool и DirectumRX;
  2.3. В менеджере экземпляров системы нажать кнопку "Добавить";
  2.4. Выбрать папку в которую разархивирован DirectumLauncher (п. 2.1.);
  2.5. Указать новый код системы и заполнить прочие аттрибуты экземпляра;
  2.6. Нажать кнопку "Установить", дождаться завершения установки и нажать кнопку повторно;
  2.7. Перевести режим запуска службы на "Вручную", иначе после перезагрузки все службы запустятся автоматически.
3. После удаления инстанса необходимо:
  3.1. Вручную удалить: сайт, пул приложений, базу данных;
  3.2. Если была установлена служба сервис раннера, перезагрузить компьютер.
